def stmspi.init_bus (   bus_num,

Turn on an SPI bus or reinitialize it if it was on already.

  • bus_num (int): Must be 1-4. Selects the SPI bus.
  • baudrate (int): Speed of transmission. Default is 4000000.
  • polarity (int): Must be 0 or 1. The idle level for the clock line.
  • phase (int): Must be 0 or 1. Sample data on the first or second clock edge, respectively.
  • firstbit (int): Use 'LSB' or 'MSB' for least or most significant bit first, respectively.
Returns
0 if the bus was successfully (re)initialized.
-1 if there was an argument error.
