def stm_main.MotorTask.run_task (   self,
  cmd_code = 'init' 
)

The state machine for the MotorTask.

Run this once per loop and update the argument from there.

The command code can be one of the following:

  • init (re-)initialize this MotorTask.
  • slew # Go to a position, in absolute degrees.
  • turn # Go to a position, in relative degrees.
  • track Turn at a constant rate.
  • mark [set] Go to the MARK position [set the current position as MARK].
  • home [set] Go to the HOME position [set the current position as HOME].
  • stop Stop the motor, with a holding torque.
  • off Set the motor driver to Hi-Z (coast) mode.
  • cmd_code A string that represents the requested instruction.
Returns
error The error code. 0 if no error.
